foody is a blog which contains recepies for cheap and easy prepared food. 
foody has 8 different recepies, which users can comment on each. 
foody also has an index and an about page. 


Technologies
----

* HTML5
* CSS3
* XML
* PHP
* JavaScript
* Google reCAPTCHA 

We have used HTML5 to view the content to the reader. 
CSS3 to style and make the site responsive. 
XML to store comments. 
PHP to recieve comments and store them in a XML file. 
JavaScript to read the already existing comments from the XML file. 
Google reCAPTCHA to avoid spam. This is the only library we have used and we have included the PHP implementation.
